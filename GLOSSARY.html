TPP

Timing Post-Processor, a built-in feature of Aegisub that can help automate some timing work, but doesn't always yield desired results.

kfx

Karaoke effects, often in softsubs represented with {\k} tags.

Dependency Control

An automation script package manager built by fansubbers. The open-source repository can be found at https://github.com/TypesettingTools/DependencyControl.
